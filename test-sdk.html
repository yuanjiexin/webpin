WebPin — 网页批注工具

这是一个 SDK 嵌入演示页面。右下角的 ✏️ 按钮可激活批注模式，点击页面上任意元素即可添加批注。

功能介绍

WebPin 是一个类似 markup.io 的网页批注工具，支持：

  • ✏️ 点选任意页面元素添加批注
  • 👤 记录批注人信息（姓名、时间）
  • 💬 回复和讨论线程
  • ✅ 标记批注为已解决
  • 🎨 彩色批注标记

SDK 安装方法

在你的网页 <body> 末尾添加：

<script src="https://your-domain.com/embed.js" data-project-id="YOUR_PROJECT_ID" data-api-key="YOUR_API_KEY" ></script>

Chrome 扩展安装

通过 Chrome 扩展，你可以对任意网页（包括不是你自己的网站）添加批注，适合设计评审、网页反馈等场景。
